왜 프롬프트를 외울 필요가 없는가?
15분

가장 많이 받는 질문

"프롬프트를 어떻게 잘 써야 하나요?"

프롬프트를 외울 필요 없습니다.

AI가 프롬프트를 만들어 주기 때문입니다.

기존 방식 vs AI닷 방식
❌ 기존 방식 (어려운 방법)

사람이 프롬프트를 공부하고
외운 후 직접 작성
→ 시간 소요, 잊어버림, 어려움

✅ AI닷 방식 (쉬운 방법)

내 문제를 AI에게 설명하고
프롬프트 생성을 요청
→ 즉시 활용, 반복 가능, 쉬움

실제 입력 예시
❌ 잘못된 접근
사용자 입력

"회의록 만들어줘"

→ 문제: 너무 모호해서 AI가 어떤 회의록을 원하는지 알 수 없음

✅ 올바른 접근 — AI닷 방식 (3단계)
1
문제 설명
"저는 팀장인데 매주 월요일 오전 팀 회의 후 회의록을 작성하는데 30분 이상 걸립니다."
2
프롬프트 생성 요청
"이 상황을 해결할 고급 회의록 자동화 프롬프트를 만들어 주세요."
3
AI가 전문 프롬프트 생성
→ AI가 상황에 맞는 최적의 프롬프트를 자동으로 완성합니다.
🎯 강사 실습 포인트

지금 바로 ChatGPT나 Claude를 열고 다음을 입력해 보세요:

"나는 [내 직업]이고, [내 문제]가 있습니다. 이 문제를 해결할 프롬프트를 만들어 주세요."

AI 문제해결 5단계 프레임워크
20분
전체 구조 한눈에 보기
단계 내용 핵심 질문
STEP 1 문제 발견 어떤 업무에서 불편함/비효율이 있는가?
STEP 2 문제 상세 설명 언제, 누가, 어떤 상황에서, 얼마나 걸리는가?
STEP 3 프롬프트 제작 요청 이 문제를 해결할 프롬프트를 만들어 주세요
STEP 4 AI 결과 생성 및 확인 AI가 만든 결과가 내 상황에 맞는가?
STEP 5 검토 → 수정 → 최종 활용 사실 확인, 보완, 실무 적용
STEP 2 — 문제 상세 설명 공식 (STRO)
S
Situation (상황)
나는 [직업/직급]이고, [업무 환경]에서 일합니다
T
Task & Time (과제/시간)
[어떤 작업]을 [얼마나 자주/오래] 하고 있습니다
R
Result (원하는 결과)
[어떤 결과물]이 필요합니다
O
Output Format (출력 형식)
[표/문서/요약/리스트] 형태로 출력해 주세요

실전 사례 심층 분석 3가지
25분 · 사례당 약 8분
1
회의록 자동화
팀장 · 직장인 · 주 3회 회의
업무 상황팀장급 직원 / 매주 3회 팀 회의 / 회의 후 회의록 작성
기존 문제회의 내용을 수기로 정리 → 1회당 30~40분 소요 / 주 3회 = 주당 최소 90분 낭비
목표회의록 작성 시간을 5분 이내로 단축
적용 AIChatGPT, Claude, Google Gemini 중 선택
① 사용자가 AI에게 입력하는 문장 (STRO 공식 적용)
"저는 IT 회사의 개발팀 팀장입니다. 매주 월/수/금 오전 9시에 팀 전체 회의가 있는데, 회의 후 회의록 작성에 30~40분이 걸립니다. 회의 내용을 텍스트로 붙여넣으면 자동으로 회의록을 만들어주는 프롬프트를 작성해 주세요. 결과물은 표 형태로 회의 목적, 논의 사항, 결정 사항, 담당자, 마감일이 포함되어야 합니다."
② AI가 자동 생성하는 회의록 프롬프트
당신은 전문 비서입니다.

제공된 회의 내용을 분석하여 아래 형식의 회의록을 표 형태로 작성하세요.

1. 회의 일시 및 참석자
2. 회의 목적
3. 주요 논의사항 (번호 순)
4. 결정사항
5. Action Item (담당자 + 마감일)
6. 다음 회의 예정 안건
③ 기대 효과
  • 시간 절감: 주 90분 → 주 15분 (약 83% 절감)
  • 연간 환산: 약 78시간 절감 (약 2주 근무 시간)
  • 품질 향상: 누락 없는 일관된 회의록 형식
  • 팀원 공유: 즉시 이메일/슬랙 공유 가능
2
매출 감소 원인 분석 보고서
중소기업 영업관리자 · 임원 보고용
업무 상황중소기업 영업관리자 / 월별 매출 데이터 보유
기존 문제매출 감소 원인 분석 + 보고서 작성 = 반나절 이상 소요
데이터 현황1월 1,500만원 → 2월 1,300만원 → 3월 1,100만원 (3개월 연속 감소)
목표임원 보고용 분석 보고서를 30분 내에 완성
적용 AIChatGPT, Claude, Google Gemini 중 선택
① 사용자가 AI에게 입력하는 문장 (STRO 공식 적용)
"저는 중소기업 영업팀 관리자입니다. 최근 3개월(1월 1,500만원 → 2월 1,300만원 → 3월 1,100만원)간 매출이 연속 감소하고 있습니다. 이 데이터를 바탕으로 매출 감소 원인을 분석하고 임원 보고용 보고서를 만들어주는 프롬프트를 작성해 주세요. 결과물은 현황 요약, 원인 분석, 개선 방안, 기대 효과 순서로 표 형태로 출력해 주세요."
② AI가 자동 생성하는 매출 분석 프롬프트
당신은 경영 컨설턴트입니다.

제공된 매출 데이터를 분석하여 아래 형식의 경영 보고서를 표 형태로 작성하세요.

1. 현황 요약 (3개월 매출 추이 및 증감률)
2. 문제점 분석 (감소 패턴 식별 및 원인 가설)
3. 원인 진단 (내부 요인 / 외부 요인 분류)
4. 개선 방안 (단기 3가지 / 중기 2가지)
5. 실행 계획 (담당자 + 일정 + 예산 + KPI)
6. 기대 효과 (수치 기반 개선 효과 예측)
③ 기대 효과
  • 시간 절감: 반나절(4시간) → 30분 이내 (약 87% 절감)
  • 보고서 품질: 6개 섹션 구조화 보고서 자동 완성
  • 의사결정 속도: 임원 보고까지 당일 처리 가능
  • 반복 활용: 매월 데이터만 교체하면 즉시 재사용
3
공공기관 민원 분석 보고서
시청/구청/공공기관 민원 담당 공무원
업무 상황시청/구청/공공기관 민원 담당 공무원
기존 문제월별 민원 데이터를 엑셀로 정리 후 보고서 작성 → 3~4시간 소요
데이터 현황전화 민원 350건, 방문 민원 120건, 온라인 민원 89건 (전월 대비 각각 +15%, +8%, +22%)
목표민원 원인 분석 + 개선안 포함 보고서를 1시간 내 완성
적용 AIChatGPT, Claude, Google Gemini 중 선택
① 사용자가 AI에게 입력하는 문장 (STRO 공식 적용)
"저는 구청 민원 담당 공무원입니다. 이번 달 전화 민원 350건, 방문 민원 120건, 온라인 민원 89건이 접수되었고, 전월 대비 각각 15%, 8%, 22% 증가했습니다. 민원 증가 원인을 분석하고 개선 방안을 포함한 내부 보고서를 작성해주는 프롬프트를 만들어 주세요. 결과물은 민원 현황, 유형별 분류, 원인 분석, 개선 방안, 예산 절감 효과 순서로 표 형태로 출력해 주세요."
② AI가 자동 생성하는 민원 분석 프롬프트
당신은 공공행정 전문 분석가입니다.

제공된 민원 데이터를 분석하여 아래 형식의 공공기관 내부 보고서를 표 형태로 작성하세요.

1. 민원 현황 요약 (채널별 건수 및 전월 대비 증감률)
2. 민원 유형 분류 (민원 성격별 상위 5개 유형 정리)
3. 증가 원인 분석 (내부 요인 / 외부 요인 / 계절적 요인)
4. 개선 방안 (단기 처리 방안 3가지 / 중장기 시스템 개선안 2가지)
5. 예산 절감 효과 (온라인 전환 시 처리 비용 절감 추정액)
③ 기대 효과
  • 시간 절감: 3~4시간 → 1시간 이내 (약 70% 절감)
  • 보고서 품질: 유형 분류 + 원인 + 개선안 자동 완성
  • 예산 효과: 온라인 전환 유도 시 처리 비용 절감 근거 확보
  • 정책 반영: 민원 데이터 기반 시정 개선안 즉시 보고 가능
